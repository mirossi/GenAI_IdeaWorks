Instruction:
1) install requirements with pip:  "pip install -r requirements.txt"
2) run with streamlit: "streamlit run app.py"

Demo:
- you will find a working demo here: https://genaiideaworks.streamlit.app/
- you will need an openAI key to run the demo: 
  - get your openAI key at https://openai.com/blog/openai-api 
  - start the demo and enter your openAI key in the side-bar within the app
